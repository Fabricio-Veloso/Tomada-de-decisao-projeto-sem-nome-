# Catalogo inicial de dinamicas

## Papel do banco de dinamicas

O banco de dinamicas existe para reunir dinamicas capazes de potencializar a eficiencia dos nos.

## Tipos ja identificados

- dinamicas comuns
- dinamicas experimentais
- dinamicas base

## Direcao de documentacao

Cada dinamica que se queira tornar reutilizavel deve ter um DPD correspondente.
