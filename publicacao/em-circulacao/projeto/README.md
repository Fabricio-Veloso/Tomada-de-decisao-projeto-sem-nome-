# DFIs em circulacao deste projeto

Esta pasta guarda DFIs sobre o proprio projeto de melhoria da tomada de decisao.

Use esta area quando o documento estiver tratando de conceitos, regras, fluxos, artefatos ou criterios internos deste projeto.

## Como usar

- coloque aqui DFIs que ja podem circular
- mantenha o estado do fluxo visivel no proprio documento
- registre origem e documentos relacionados quando existirem
- mova ou atualize documentos quando forem consolidados ou arquivados

Para o fluxo completo, veja o [Guia operacional para publicar DFIs](../../guia-operacional-para-publicar-dfis.md).
